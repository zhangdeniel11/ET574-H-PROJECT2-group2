Math Functions Library
📌 Overview

This project implements a collection of core and original mathematical functions in Python. Each function is designed with correctness, efficiency, and proper handling of edge cases in mind. Unit tests are included to verify functionality.

🧮 Functions Implemented
Core Functions
factorial(x)
Computes the factorial of a non-negative integer.
sqrt(x)
Computes the square root of a non-negative number using an approximation method (no math module used).
abs_val(x)
Returns the absolute value of a number.
